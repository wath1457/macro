정보저장 파일
1. 차량번호 렌트카업체 부과번호
2. 차량번호 렌트카업체 부과번호
3. ...


임차인 변경 순서
1단계 : 트래비스에서 운전자변경 (사용자)
2단계 : 부과취소 (컴퓨터)
3단계 : 행정포털 (컴퓨터)


*부과취소
부과번호 읽기 -> 부과번호 칸 클릭, 입력 -> 조회(F3) -> 내역 더블클릭 -> 부과취소 클릭 -> 예 클릭 -> 확인 클릭 -> 종결일자 클릭, 오늘날짜 입력 -> 종결사유 클릭, 입력 -> 저장 클릭 -> 확인 클릭 -> 창 닫기 클릭

@행정포털

*문서등록

비전자문서 클릭 -> 접수 비전자문서 등록 클릭 -> 제목 클릭, 차량번호 읽기, 제목입력 -> 발신기관명 클릭, 렌트카업체 읽기, 발신기관입력 -> 공개여부 클릭 -> 비공개 클릭 -> 비공개근거 3호 6호 클릭 -> 비공개사유 클릭, 입력 -> 접수 클릭 -> 확인 클릭 - ctrl + f 류찬아 검색 -> 이름 더블클릭, 확인

*결재

제목에 '렌트카 부과 변경 (' 검색 -> 모두선택 클릭 -> 일괄결재 클릭 -> 과제등록 클릭 -> 주정차과태료 클릭 -> 일괄처리 클릭


사용자가 해야할것
1. 임차인 1단계
2. 정보파일에 차량번호, 렌트카업체 이름, 부과번호 입력
3. 3단계 도장찍고, 번호매기기(펜)

필요함수
1. 마우스 클릭 함수(좌표)
2. 키 입력 함수(동시입력 가능해야함) ctrl+f,F3 
3. 정보파일 읽기 함수



고안
1. 내용 입력 부분은 미리만들어놓은 텍스트파일에서 읽어온다.
2. 키입력은 mouse_event와 keybd_event 사용
3. 사용자가 부과번호를 메모장에저장(엔터로 구분) -> 메모장에 저장된 부과번호를 읽고 또다른메모장에 대장관리에서 딴 차번호와 렌트카업체명을 저장 (복사 붙여넣기 반복, 단! 이때 마우스커서는 항상 메모장 우측최하단에 위치하여 클릭한다 올바른 복사위치 때문에) -> 이방법은 고려필요
4. '부과번호 렌트카업체' 이 양식에서 부과번호는 공백문자가 나올때까지 읽고 렌트카업체는 엔터가 나올때까지 읽는다 fin.getline() or get에서 문자 하나 대조하기


최종안
1.사용자가 부과번호를 메모장에 직접입력한다.
2.fin을통해 부과번호를 읽고 부과번호로 검색, 대장관리에 들어간다
3.대장관리에서 차번호를 복사한다.(클립보드에 복사)
4.클립보드 붙여넣기를 통해 임시기억공간(배열)에 저장한다.
5. 3,4번을 렌트카업체도 똑같이 반복한다
6. 새로운 메모장에 fout을 통해 차량번호와 렌트카업체를 쓴다. -> 2단계 완

7. 메모장 내용을 fin을통해 읽는다
8. 클립보드 복사를하고 문서등록할때 붙여넣기 한다.
